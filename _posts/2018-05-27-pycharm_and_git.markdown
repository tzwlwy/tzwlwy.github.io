**安装准备**
<pre><code>
首先安装好git 和pycharm

然后新建一个项目名字，在项目文件夹下输入git init

然后在输入  git clone + 地址

这样一个项目就生成好了

然后在pycharm--file--settings--version control

里面绑定git 和git hub就可以了
</code></pre>
